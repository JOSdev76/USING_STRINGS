Cambio de Mayúsculas y Minúsculas

Métodos para transformar entre mayúsculas y minúsculas.

.lower()

Para qué sirve: Devuelve una copia del string con todos los caracteres en minúsculas.

Parámetros: Ninguno.
Ejemplo: 
texto = "HOLA MUNDO"
print(texto.lower())  # 'hola mundo'

.upper()

Para qué sirve: Devuelve una copia con todos los caracteres en mayúsculas.

Parámetros: Ninguno.
Ejemplo: 
texto = "hola mundo"
print(texto.upper())  # 'HOLA MUNDO'

.capitalize()

Para qué sirve: Devuelve una copia con el primer carácter en mayúscula y el resto en minúsculas.

Parámetros: Ninguno.
Ejemplo: 
texto = "hola. ¿QUÉ TAL?"
print(texto.capitalize())  # 'Hola. ¿qué tal?'

.title()

Para qué sirve: Devuelve una copia donde el primer carácter de cada palabra está en mayúscula.

Parámetros: Ninguno.
Ejemplo: 
texto = "el señor de los anillos"
print(texto.title())  # 'El Señor De Los Anillos'

.swapcase()

Para qué sirve: Devuelve una copia con mayúsculas y minúsculas intercambiadas.

Parámetros: Ninguno.
Ejemplo: 
texto = "Python Es Genial"
print(texto.swapcase())  # 'pYTHON eS gENIAL'

.casefold()

Para qué sirve: Similar a `.lower()`, pero más agresivo para comparaciones en diferentes idiomas (ej: "ß" se convierte en "ss").

Parámetros: Ninguno.
Ejemplo: 
texto_aleman = "Straße"
print(texto_aleman.casefold()) # 'strasse'

Búsqueda y Conteo

Métodos para buscar subcadenas y contar ocurrencias.

.find(sub, start, end)

Para qué sirve: Busca `sub` y devuelve el índice de la primera aparición. Devuelve -1 si no la encuentra.

Parámetros:
  • `sub`: String a buscar.
  • `start` (Opcional): Índice de inicio.
  • `end` (Opcional): Índice de fin.
Ejemplo: 
texto = "hola mundo"
print(texto.find("mundo"))  # 5
print(texto.find("sol"))    # -1

.rfind(sub, start, end)

Para qué sirve: Igual que `.find()`, pero devuelve el índice de la *última* aparición (buscando desde la derecha).

Parámetros:
  • `sub`: String a buscar.
Ejemplo: 
texto = "hola mundo, adiós mundo"
print(texto.rfind("mundo")) # 18

.index(sub, start, end)

Para qué sirve: Igual que `.find()`, pero lanza un error (ValueError) si no encuentra `sub`.

Parámetros:
  • `sub`: String a buscar.
Ejemplo: 
texto = "hola mundo"
print(texto.index("hola"))  # 0
# texto.index("sol")  # ValueError

.rindex(sub, start, end)

Para qué sirve: Igual que `.rfind()`, pero lanza un error (ValueError) si no encuentra `sub`.

Parámetros:
  • `sub`: String a buscar.
Ejemplo: 
texto = "hola mundo, adiós mundo"
print(texto.rindex("mundo")) # 18

.count(sub, start, end)

Para qué sirve: Devuelve el número de veces que `sub` aparece en el string.

Parámetros:
  • `sub`: String a contar.
Ejemplo: 
texto = "pan, pan y más pan"
print(texto.count("pan"))  # 3

.startswith(prefix, ...)

Para qué sirve: Devuelve `True` si el string comienza con `prefix`.

Parámetros:
  • `prefix`: String o tupla de strings a comprobar.
Ejemplo: 
archivo = "documento.pdf"
print(archivo.startswith("doc")) # True

.endswith(suffix, ...)

Para qué sirve: Devuelve `True` si el string termina con `suffix`.

Parámetros:
  • `suffix`: String o tupla de strings a comprobar.
Ejemplo: 
archivo = "imagen.jpg"
print(archivo.endswith((".jpg", ".png"))) # True

Limpieza y Alineación

Métodos para limpiar espacios y alinear texto.

.strip(chars)

Para qué sirve: Elimina caracteres del inicio y del final. Por defecto, elimina espacios en blanco.

Parámetros:
  • `chars` (Opcional): String de caracteres a eliminar.
Ejemplo: 
usuario = "   admin@email.com   "
print(usuario.strip())  # 'admin@email.com'

.lstrip(chars)

Para qué sirve: Igual que `strip()`, pero solo elimina del lado izquierdo (left).

Parámetros:
  • `chars` (Opcional): Caracteres a eliminar.
Ejemplo: 
texto = "---¡Hola!"
print(texto.lstrip("-"))  # '¡Hola!'

.rstrip(chars)

Para qué sirve: Igual que `strip()`, pero solo elimina del lado derecho (right).

Parámetros:
  • `chars` (Opcional): Caracteres a eliminar.
Ejemplo: 
texto = "¡Hola!   "
print(texto.rstrip())  # '¡Hola!'

.center(width, fillchar)

Para qué sirve: Centra el string en un `width` total, rellenando con `fillchar`.

Parámetros:
  • `width`: Ancho total.
  • `fillchar` (Opcional): Carácter de relleno.
Ejemplo: 
titulo = "MENÚ"
print(titulo.center(20, "="))

.ljust(width, fillchar)

Para qué sirve: Justifica a la izquierda (padding a la derecha).

Parámetros:
  • `width`: Ancho total.
Ejemplo: 
print("Texto".ljust(10, "-")) # 'Texto-----'

.rjust(width, fillchar)

Para qué sirve: Justifica a la derecha (padding a la izquierda).

Parámetros:
  • `width`: Ancho total.
Ejemplo: 
print("Texto".rjust(10, "-")) # '-----Texto'

.zfill(width)

Para qué sirve: Rellena a la izquierda con ceros (`0`) para alcanzar el `width` total.

Parámetros:
  • `width`: Ancho total.
Ejemplo: 
print("5".zfill(4))  # '0005'
print("-12".zfill(6)) # '-00012'

.expandtabs(tabsize)

Para qué sirve: Reemplaza los caracteres de tabulación (`\t`) por espacios.

Parámetros:
  • `tabsize` (Opcional): Número de espacios (8 por defecto).
Ejemplo: 
texto = "col1\tcol2"
print(texto.expandtabs(4)) # 'col1    col2'

División y Unión

Métodos para dividir strings y unir elementos.

.split(sep, maxsplit)

Para qué sirve: Divide el string en una lista usando `sep` como separador. Si `sep` se omite, divide por espacios.

Parámetros:
  • `sep` (Opcional): Separador.
  • `maxsplit` (Opcional): Máximo de divisiones.
Ejemplo: 
frase = "Hola,mundo"
print(frase.split(",")) # ['Hola', 'mundo']

.rsplit(sep, maxsplit)

Para qué sirve: Igual que `split()`, pero comienza a dividir desde la derecha (solo relevante con `maxsplit`).

Parámetros:
  • `maxsplit` (Opcional): Máximo de divisiones.
Ejemplo: 
linea = "a,b,c,d"
print(linea.rsplit(",", 2)) # ['a,b', 'c', 'd']

.splitlines(keepends)

Para qué sirve: Divide el string en una lista por los saltos de línea (`\n`, `\r`, etc.).

Parámetros:
  • `keepends` (Opcional): `True` para mantener los saltos.
Ejemplo: 
parrafo = "Línea 1\nLínea 2"
print(parrafo.splitlines()) # ['Línea 1', 'Línea 2']

.partition(sep)

Para qué sirve: Busca `sep` y devuelve una tupla de 3: (antes, separador, después).

Parámetros:
  • `sep`: Separador a buscar.
Ejemplo: 
email = "usuario@dominio.com"
print(email.partition("@"))
# ('usuario', '@', 'dominio.com')

.rpartition(sep)

Para qué sirve: Igual que `partition()`, but busca la *última* aparición de `sep` (desde la derecha).

Parámetros:
  • `sep`: Separador a buscar.
Ejemplo: 
path = "/usr/bin/python"
print(path.rpartition("/"))
# ('/usr/bin', '/', 'python')

.join(iterable)

Para qué sirve: Une los elementos de un iterable (ej: lista) en un solo string, usando el string como "pegamento".

Parámetros:
  • `iterable`: Lista de strings a unir.
Ejemplo: 
palabras = ['Hola', 'mundo']
print(" ".join(palabras)) # 'Hola mundo'

.replace(old, new, count)

Para qué sirve: Reemplaza todas las ocurrencias de `old` por `new`.

Parámetros:
  • `old`: String a reemplazar.
  • `new`: String nuevo.
  • `count` (Opcional): Máximo de reemplazos.
Ejemplo: 
texto = "pan, pan y más pan"
print(texto.replace("pan", "agua", 2))
# 'agua, agua y más pan'

Verificación de Tipo

Métodos que devuelven True/False según el contenido del string.

.islower()

Para qué sirve: `True` si todos los caracteres con "caja" (letras) son minúsculas.

Ejemplo: 
print("hola".islower()) # True

.isupper()

Para qué sirve: `True` si todos los caracteres con "caja" son mayúsculas.

Ejemplo: 
print("HOLA".isupper()) # True

.istitle()

Para qué sirve: `True` si el string tiene formato de título (cada palabra empieza con mayúscula).

Ejemplo: 
print("El Señor".istitle()) # True

.isalpha()

Para qué sirve: `True` si todos los caracteres son letras (sin espacios, números o símbolos).

Ejemplo: 
print("Python".isalpha()) # True
print("Python 3".isalpha()) # False

.isdigit()

Para qué sirve: `True` si todos los caracteres son dígitos (0-9).

Ejemplo: 
print("12345".isdigit()) # True
print("123.45".isdigit()) # False

.isdecimal()

Para qué sirve: Similar a `isdigit()`, pero más estricto (solo para caracteres decimales).

Ejemplo: 
print("123".isdecimal()) # True

.isnumeric()

Para qué sirve: Más amplio que `isdigit()`. `True` si todos son numéricos (incluye fracciones de unicode, etc.).

Ejemplo: 
print("123".isnumeric()) # True

.isalnum()

Para qué sirve: `True` si todos los caracteres son alfanuméricos (letras o números).

Ejemplo: 
print("Usuario10".isalnum()) # True
print("Usuario 10".isalnum()) # False

.isspace()

Para qué sirve: `True` si el string solo contiene espacios en blanco (espacio, `\t`, `\n`, etc.).

Ejemplo: 
print(" \n\t ".isspace()) # True

.isidentifier()

Para qué sirve: `True` si el string es un identificador válido de Python (nombre de variable).

Ejemplo: 
print("mi_variable".isidentifier()) # True

.isprintable()

Para qué sirve: `True` si todos los caracteres son "imprimibles" (no incluye `\n` o `\t`).

Ejemplo: 
print("Hola".isprintable()) # True
print("Hola\n".isprintable()) # False

Formato de Strings

Métodos para formatear strings con variables y valores.

f-strings (Recomendado)

Para qué sirve: La forma moderna y preferida de insertar variables en strings. Se usa poniendo una `f` antes de las comillas.

Ejemplo: 
nombre = "Carlos"
edad = 30
mensaje = f"Hola, {nombre}, tienes {edad} años."
print(mensaje)

.format()

Para qué sirve: El método más antiguo para formatear strings, usando llaves `{}` como marcadores de posición.

Ejemplo: 
nombre = "Carlos"
edad = 30
mensaje = "Hola, {}, tienes {} años.".format(nombre, edad)
print(mensaje)

.format_map(mapping)

Para qué sirve: Similar a `.format()`, pero usa un diccionario para los valores.

Ejemplo: 
datos = {"nombre": "Ana", "edad": 25}
mensaje = "Nombre: {nombre}".format_map(datos)
print(mensaje) # 'Nombre: Ana'
